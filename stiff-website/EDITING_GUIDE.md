# 편집 가이드

## 🎯 주요 편집 포인트

### 1. 텍스트 내용 변경
**파일**: `index.html`
- 메인 타이틀: "We make Laughs That look Damn Good" 부분
- 회사 정보: "Stiff Films", "Baltimore, MD" 부분
- 팀 멤버 정보: Matt Pittroff, Brenna Mathers, Steve Blair 섹션

### 2. 스타일 커스터마이징
**파일**: `css/styles-custom.css`
- 색상 변경
- 폰트 크기 조정
- 레이아웃 수정
- 애니메이션 속도 조절

### 3. JavaScript 기능 수정
**파일**: `js/main-stiff.js`
- 애니메이션 타이밍
- 인터랙션 기능
- 스크롤 동작

## 🎨 주요 CSS 클래스들

- `.f-80`, `.f-180`, `.f-240`: 폰트 크기 클래스
- `.Red`: 빨간색 텍스트
- `.hero`: 메인 히어로 섹션
- `.work-box`: 작품 박스
- `.split.chars`: 텍스트 애니메이션용

## 🖼️ 이미지 교체

이미지를 교체하려면:
1. `img/` 폴더에 새 이미지 업로드
2. `index.html`에서 해당 이미지 경로 수정
3. 동일한 파일명으로 교체하면 경로 수정 불필요

## 🎬 애니메이션 수정

### Lottie 애니메이션
- `data-lottie` 속성으로 제어
- `js/main-stiff.js`에서 재생 로직 수정

### GSAP 애니메이션
- ScrollTrigger를 사용한 스크롤 애니메이션
- SplitText를 사용한 텍스트 애니메이션

## 🔧 개발 팁

1. **라이브 리로드**: Live Server 확장 사용 권장
2. **디버깅**: 브라우저 개발자 도구 활용
3. **모바일 테스트**: 반응형 디자인 확인
4. **성능**: 이미지 최적화 및 압축

## 📱 반응형 디자인

- `ws-xl-*`: 데스크톱 레이아웃
- `ws-sm-*`: 태블릿 레이아웃  
- `ws-xs-*`: 모바일 레이아웃
